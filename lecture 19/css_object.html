Object ft and Object position

object-fit: cover // this means that the image with width and height can in the area without losing its aspect ratio but in this some part of image may be cut off or crop

object-fit: contain // this means that the image does not lose its aspect ratio but may be some part of the width and height is left empty

object-position: top right // this is used when some part of image is cut during object-fit: cover so if we want some specific part of the image that is more important or for any reason then we have to use this or something like bottom left or center center or top left etc
