The specificity depends in following rate !important > inline > id selector > class or attribute or pseudo selector > element selector > universal selector

The value of specificity for each selector is given below :

Universal selector: 0,      e.g *

Element selector: 1,      e.g <p></p>tag, <a></a>tag, etc

Class, attribute, pseudo selector : 10,      e.g .box, a[href], a:link etc

Id selector : 100,      e.g #redbox

Inline styling: 1000,      e.g style="color:purple;"

!important : 10000,      e.g background-color: grey ! important;

The specificity of p.box.redbox p:first-child [href] is 1 + 10 + 10 + 10 + 10 = 41
Note: If the specificity of both are same then the styling apply by position
