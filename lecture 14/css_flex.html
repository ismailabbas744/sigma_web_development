flex

properties apply on container

  1. display: flex // use to arrange items inline
  2. justify-content: center // use to center from left to right if flex-direction: row and vise-vers
  3. align-items: center // use to center from top to bottom if flex-direction: row and vise versa
  4. flex-direction: row // use to arrange in row it may be column
  5. gap: 40 //use to add space or gap
  6. align-content: center // use to align content i.e all the content center
  7. flex-wrap: wrap // use to wrap items i.e if the with of the container is full then the new items arrange in next line or row

Properties apply on item

    1. order:1 // by default the order of each element is zero so if we apply order of 1 then if goes to the end .it can be 2, 3, 4 ....
    2. flex-grow: 4 // it means that this element can take the space of 4 items
    3. flex-shrink: 2 // this means that if the width decreases and there is no space left then first of all this element shrinks to its half size and then all the other including this element shrinks accordingly
    4. align-self: flex-start // this means that this element align itself to the start of the flex
