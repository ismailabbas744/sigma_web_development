width: 100vw:

This means that the width is equal to the 100% of viewports width e.g for desktop it is 100% of desktop width and similarly 100% for mobile and so on.


width: 100vh:

This is same as above but this is 100% fo viewport height


width: 1.5em:

This means that the width is equal to the 1.5x of the width given to it by parent e.g if parent width:100px then the width: 1.5em means its width:150px i.e 1.5(100) = 150


width: 1.5rem

This means that its width is 1.5x of the width of the root e.g if html{ width: 100px} then its width:150px


width: 100vmin

This means that its width is equal to the 100% of the viewport minimum i.e in laptop height is less than width so it is 100 of width of the laptop. And in mobile the vmin is width i.e width of mobile is less than height so it is 100% of the width of the mobile
